# AgentFlow AI

AgentFlow AI is a LangGraph-powered AI Agent built using FastAPI, LangChain, OpenAI, and ChromaDB.

Unlike traditional RAG chatbots that always follow a fixed retrieval → answer pipeline, AgentFlow AI uses graph-based workflows and decision-making nodes to determine the best path for answering user queries.

## Features

* LangGraph State-Based Workflows
* AI Agent Architecture
* Conditional Routing
* Tool Calling
* Document Retrieval
* ChromaDB Vector Search
* OpenAI Integration
* Redis Memory
* FastAPI Backend
* Streaming Responses
* Extensible Agent Design

---

## Architecture

User Query
↓

↓

↓

↓

↓

↓

↓
Final Answer
